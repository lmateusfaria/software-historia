# Checklist de Funcionalidades do Sistema

Este checklist foi criado para validar as funcionalidades do sistema. Marque com um "X" as funcionalidades que estão prontas e funcionais.

---

## Backend

- [X] **Cadastro de Usuários**: O sistema permite criar novos usuários.
- [X] **Autenticação**: O sistema realiza login e gera tokens JWT.
- [X] **Atualização de Usuários**: O sistema permite editar informações de usuários.
- [X] **Exclusão de Usuários**: O sistema permite excluir usuários.
- [X] **Listagem de Usuários**: O sistema retorna informações detalhadas de usuários.
- [X] **Entradas e Saídas**: O sistema registra transações financeiras (entradas e saídas).
- [ ] **Relatórios**: O sistema gera relatórios financeiros detalhados.

---

## Frontend

### Funcionalidades Gerais
- [ ] **Dashboard**: A página inicial exibe informações resumidas (saldo, contas, lançamentos).
- [ ] **Responsividade**: O layout se adapta a diferentes tamanhos de tela.

### Usuários
- [X] **Exibição de Dados**: O modal exibe corretamente os dados do usuário logado.
- [X] **Edição de Dados**: O modal permite editar os dados do usuário logado.
- [X] **Bloqueio de Campos**: Os campos do modal estão bloqueados inicialmente e só são liberados ao clicar em "Alterar".
- [X] **Exclusão de Usuário**: O botão "Excluir" remove o usuário logado.

### Transações
- [X] **Registro de Transações**: O sistema permite registrar novas transações (entradas e saídas).
- [ ] **Listagem de Transações**: O sistema exibe uma lista de transações recentes.

### Relatórios
- [ ] **Geração de Relatórios**: O sistema exibe relatórios financeiros detalhados.
- [ ] **Gráficos**: Os relatórios incluem gráficos para visualização de dados.

---

Marque as funcionalidades concluídas à medida que forem validadas. Se encontrar problemas, registre os detalhes para correção.
